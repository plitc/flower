=head1 DESCRIPTION

PerlPeer is a meshed P2P client running between trusted peers on a LAN
or WAN network.

=head1 STATUS

PerlPeer is not yet operational.

=head1 SYNOPSIS

  git clone git://github.com/tardisx/PerlPeer.git
  cd PerlPeer
  perl Makefile.PL
  make
 mojo get -M POST -c 'bob' 172.22.99.165:2222/REST/1.0/ping
 Then visit L<http://127.0.0.1:2222> in your browser.
